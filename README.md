# Tranquil (static)

A tiny static website that helps you:

- Download the Tranquil gray background image (so you can set it as your iPhone wallpaper)
- Add a matching gray Home Screen icon via “Add to Home Screen”

## Deploy to Cloudflare Pages

1. Create a Cloudflare Pages project from this repo
2. Framework preset: **None**
3. Build command: *(leave empty)*
4. Build output directory: `.`

## Local preview

Open `index.html` directly, or serve the folder with any static server.
